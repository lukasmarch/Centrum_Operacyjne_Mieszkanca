## Kluczowe funkcje

### 🤖 AI Analiza (Gemini Vision)
- Automatyczna kategoryzacja zgłoszeń (8 kategorii)
- Ocena severity (low/medium/high/critical)
- Wykrywanie spamu
- Prompt z kontekstem lokalnym (jeziora, lasy, drogi Gminy Rybno)
- **Keyword Safety-Net** — zapasowe sprawdzanie 50+ polskich słów kluczowych, nawet jeśli AI się myli

### 🗺️ Mapa Leaflet
- Pinezki SVG w kształcie kropli (📍) kolorowane wg severity:
  - 🔴 Czerwona (42px) = Krytyczny
  - 🟠 Pomarańczowa (36px) = Wysoki
  - 🟡 Żółta (30px) = Średni
  - 🟢 Zielona (26px) = Niski
- Emoji kategorii wewnątrz pinezki
- Popup z tytułem, kategorią, severity, liczbą potwierdzeń

### 📍 Geolokalizacja
- Automatyczna detekcja GPS z przeglądarki
- Dropdown z 28 miejscowościami gminy Rybno + okolice
- **Wybór miejscowości ZAWSZE nadpisuje GPS przeglądarki** (naprawiony bug)
- Reverse geocoding przez Nominatim

### 📊 Panel statystyk
- Dynamiczne liczniki: Wszystkie, 🚨 Alarm, 🔥 Pożar, 🗺️ Na mapie
- Filtry kategorii i sortowanie (najnowsze/popularne)

### 🔧 Endpoint naprawczy
- `POST /api/reports/fix-existing` — naprawia istniejące raporty:
  - Kategorie z keyword matching (bez API Gemini)
  - GPS z nazwy miejscowości (nadpisuje błędne GPS przeglądarki)
