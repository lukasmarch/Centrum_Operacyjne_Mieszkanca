# Centrum Operacyjne Mieszkańca - Kontekst Projektu

## Cel projektu
Inteligentny agregator lokalnych wiadomości dla Powiatu Działdowskiego (60k mieszkańców).
Automatycznie zbiera, kategoryzuje i podsumowuje informacje z lokalnych źródeł.

## Stack technologiczny

### Backend (Python 3.11+)
- **Framework:** FastAPI + uvicorn
- **Database:** PostgreSQL 16 + SQLModel/SQLAlchemy
- **Cache:** Redis 7
- **AI:** OpenAI GPT-4o-mini (kategoryzacja), GPT-4o (podsumowania)
- **Scheduler:** APScheduler
- **Auth:** JWT (python-jose)

### Frontend (Node 18+)
- **Framework:** React 19 + TypeScript
- **Build:** Vite
- **Maps:** Leaflet + react-leaflet
- **Charts:** Recharts
- **AI:** Google Gemini (chatbot)

### Infrastruktura
- Docker Compose (PostgreSQL, Redis, Adminer)
- Alembic (migracje DB)

## Konwencje kodowania

### Python (Backend)
- **Style:** PEP 8 + Black formatter
- **Async:** Preferuj async/await dla I/O
- **Typing:** Type hints wymagane
- **Models:** Pydantic v2 dla walidacji
- **DB:** SQLModel dla ORM

### TypeScript (Frontend)
- **Style:** ESLint + Prettier
- **Components:** Functional components + hooks
- **State:** React Context (Auth, Cache)
- **API calls:** Custom hooks (useArticles, useWeather, etc.)

### Nazewnictwo
- **Pliki Python:** snake_case.py
- **Pliki TS/React:** PascalCase.tsx (komponenty), camelCase.ts (utils)
- **Zmienne:** snake_case (Python), camelCase (TypeScript)
- **Stałe:** UPPER_SNAKE_CASE

## Środowisko

### Wymagane zmienne (.env)
```
DATABASE_URL=postgresql+asyncpg://...
OPENAI_API_KEY=...
OPENWEATHER_API_KEY=...
JWT_SECRET_KEY=...
RESEND_API_KEY=...
APIFY_API_KEY=...  # opcjonalne
```

### Uruchomienie dev
```bash
# Backend
cd backend && source ../.venv/bin/activate
uvicorn src.api.main:app --reload --port 8000

# Frontend
cd frontend && npm run dev

# Docker (DB)
docker-compose up -d
```
